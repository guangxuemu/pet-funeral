### 2.3 Title与文章标题不一致检查

**问题定义：**
- frontmatter中的 `title:` 与文章正文中的 `# 标题` 不一致
- 可能导致搜索引擎困惑，影响排名

**检查方法：**
```bash
# 检查所有md文件的title和h1标题
for f in $(find docs -name "*.md"); do
  frontmatter_title=$(grep "^title:" "$f" | head -1 | sed 's/title: //' | tr -d '"')
  h1_title=$(grep "^# " "$f" | head -1 | sed 's/^# //')
  if [ "$frontmatter_title" != "$h1_title" ]; then
    echo "文件: $f"
    echo "  frontmatter: $frontmatter_title"
    echo "  H1标题: $h1_title"
    echo "---"
  fi
done
```

### 12.1 搜索引擎视角

**潜在惩罚风险：**
- ❌ **不会直接触发惩罚**：Google不会仅因title与H1不一致就惩罚网站
- ⚠️ **可能降低排名信号**：分散了页面的主题相关性信号
- ⚠️ **影响CTR**：SERP显示title，用户期望看到相关内容，不一致可能增加跳出率

**算法影响：**
| 算法 | 影响 | 说明 |
|------|------|------|
| Panda | ⚠️ 潜在影响 | 如果大量页面内容质量低或重复 |
| Helpful Content | ⚠️ 潜在影响 | 如果用户反馈差（高跳出率） |
| Core Updates | ⚠️ 间接影响 | 整体内容质量评估 |

### 12.2 不一致类型分析

| 不一致类型 | 示例 | 影响程度 | 建议 |
|-----------|------|---------|------|
| 格式差异（| vs —） | title用"|"，H1用"—" | 🟢 低 | 可接受，无需修改 |
| 标点差异 | title用"，"，H1用"、" | 🟢 低 | 可接受，无需修改 |
| 长度差异 | title含品牌名，H1不含 | 🟢 低 | 正常策略 |
| 内容差异 | title含"告别仪式"，H1无 | 🟡 中 | 建议统一核心词 |
| 完全不一致 | title和H1主题不同 | 🔴 高 | 必须修复 |

**当前站点情况：**
- 主要为**格式差异**和**标点差异**（| vs —）
- 属于🟢低风险范畴
- 但大量文件（3223个）存在此问题，建议统一
